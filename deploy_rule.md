# Deploy Rules

## To deploy on GitHub Pages
1. change base in vite.config.js to '/Dukaan-Digital/'
2. change start_url in manifest.json to '/Dukaan-Digital/'
3. change basename in App.jsx to '/Dukaan-Digital/'
4. run 'npm run deploy'

## To deploy on Vercel
1. change base in vite.config.js to '/'
2. change start_url in manifest.json to '/'
3. change basename in App.jsx to '/'
4. Just Puch the code to GitHub
5. Vercel will automatically deploy the code
