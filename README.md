## ğŸš€ **Getting Started**

### **Prerequisites**
- Node.js (v18 or higher)
- MongoDB (local or Atlas)
- Git

### **Installation**

1. **Clone the repository**
```bash
git clone https://github.com/haroon-90/Dukaan-Digital-2.git
cd Dukaan-Digital-2
```

2. **Setup Backend**
```bash
cd Backend
npm install

# Create .env file
cp .env.example .env
# Edit .env with your MongoDB URI and other configs

# Start backend server
npm start
```

3. **Setup Frontend**
```bash
cd Frontend
npm install

# Start development server
npm run dev
```

4. **Access the application**
```
Frontend: http://localhost:5173
Backend: http://localhost:5000
```
